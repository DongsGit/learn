#### 1.ArrayUtils类

~~~~java
1.ArrayUtils.toString(array);  //将数组转为为字符串，格式如：{2,4,8,16}
2.ArrayUtils.contains(array,8);  //判断数组中是否包含数字8，（一维数组)
3.ArrayUtils.indexOf(array,8);  //判断数组中第一个8的位置（一维数组)
4.ArrayUtils.lastIndexOf(array,8); //判断数组中最后一个8的位置。（一维数组)
5.ArrayUtils.clone(array)； //给一个一维数组来个拷贝
6.ArrayUtils.reverse(array);  //翻转该一维数组
7.ArrayUtils.toMap(array); //将一个二维数组转换为map，二维数组中至少与两列，第一列为key第二列为value

~~~~
